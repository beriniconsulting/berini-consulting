- Positionnement commercial clair
- Section "Ce que je peux apporter"

- Méthode d’intervention
- Outils & technologies
- Insights
- Contact

## Installation locale

1. Installer Node.js : https://nodejs.org
2. Ouvrir le dossier dans VS Code
3. Dans le terminal :

```bash
npm install
npm run dev
```

4. Ouvrir : http://localhost:3000

## Mise en ligne simple avec Vercel

1. Créer un compte GitHub
2. Créer un nouveau repository
3. Envoyer ce dossier sur GitHub
4. Aller sur https://vercel.com
5. Importer le repository
6. Cliquer sur Deploy
7. Connecter le domaine OVH ensuite
